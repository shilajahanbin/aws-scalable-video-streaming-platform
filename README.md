# AWS Scalable Video Streaming Platform

## Overview

This project presents the design of a scalable video streaming platform using AWS cloud services. The architecture focuses on high availability, low latency, scalability, and cost optimization for global video delivery.

## Technologies Used

- AWS MediaConvert
- AWS MediaPackage
- AWS CloudFront
- AWS Auto Scaling
- Elastic Load Balancing (ELB)
- AWS Lambda
- Amazon S3
- AWS CloudWatch
- AWS IAM

## Project Goals

- Deliver video content with low latency
- Support traffic spikes through automatic scaling
- Improve availability through multi-AZ architecture
- Optimize infrastructure costs
- Implement disaster recovery planning

## Architecture

The solution uses AWS Media Services for video processing and CloudFront for content delivery. Auto Scaling and Load Balancing help maintain performance during peak traffic.
